### Descripción:
En este laboratorio se trabajó de forma colaborativa utilizando Git y GitHub para implementar una clase `Calculator` en Java.  
Cada integrante desarrolló uno o más métodos en ramas independientes, aplicando operaciones básicas como suma, resta, multiplicación, división y módulo.  
Finalmente, se integraron los cambios en un repositorio común mediante el uso de merge, consolidando la solución final del proyecto.
